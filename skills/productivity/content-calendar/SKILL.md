# Content Calendar Skill

Strategic content orchestration system that transforms chaotic posting into a data-driven content engine.

## When to Use

- **Content Planning**: "Plan my content for next month"
- **Scheduling**: "Schedule this post across platforms"
- **Analytics Review**: "How did my content perform?"
- **Content Generation**: "Create a post about [topic]"
- **Audience Analysis**: "When is my audience most active?"

## Quick Reference

| Command | Description |
|---------|-------------|
| `/calendar plan [period]` | Generate content strategy |
| `/calendar schedule [post]` | Schedule publication |
| `/calendar analyze` | Performance review |
| `/calendar generate [topic]` | AI content creation |
| `/calendar optimize` | Timing recommendations |

## Procedure

### Phase 1: Strategy Development
1. Define content pillars (3-5 themes)
2. Analyze audience demographics
3. Set SMART goals (engrowth, reach, conversions)
4. Create content mix ratio (educational/promotional/entertaining)

### Phase 2: Calendar Construction
- Map content to dates/times
- Balance platform-specific formats
- Include holidays/events/trends
- Batch similar content creation tasks

### Phase 3: Execution & Tracking
- Auto-generate posts from topics
- Optimize publish times per platform
- Monitor engagement metrics
- A/B test headlines/formats

### Output Templates
- Monthly Editorial Calendar (visual table)
- Platform-Specific Post Variations
- Performance Dashboard (weekly/monthly)
- Content Recycling Schedule
