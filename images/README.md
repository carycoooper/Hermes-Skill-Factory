# 📸 收款码图片目录

## 请将收款码图片放入此目录

### 需要的文件：

1. **wechat-pay.png** - 微信收款码
2. **alipay.png** - 支付宝收款码

### 操作步骤：

1. 打开微信 → 收款 → 保存收款码图片
2. 重命名为 `wechat-pay.png`，放到本目录
3. 打开支付宝 → 收款 → 保存收款码图片
4. 重命名为 `alipay.png`，放到本目录
5. 提交到 GitHub：`git add images/ && git commit -m "Add payment QR codes"`

### 注意事项：

- ✅ 推荐使用 PNG 格式（清晰度高）
- ✅ 建议图片尺寸：400x400 像素或更大
- ⚠️ **不要**包含敏感信息（如真实姓名、完整账号）
- 🔒 这些图片将公开在 GitHub 上

### 图片示例位置：

这些图片将在 [SPONSOR.md](../SPONSOR.md) 中显示，供支持者扫码打赏。
